# FlyFood Documentation

## Project Overview
FlyFood is a food delivery application designed specifically for Tajikistan. Our mission is to provide a seamless food ordering experience, enabling users to order from their favorite local restaurants with just a few taps on their smartphones.

## Features
- **User Registration/Login:** Secure user accounts with email verification.
- **Restaurant Listings:** Browse a wide selection of restaurants categorized by cuisine types.
- **Real-Time Tracking:** Users can track their orders in real-time from the kitchen to their doorstep.
- **Payment Integration:** Multiple payment options including credit/debit cards and cash on delivery.
- **Customer Support:** 24/7 support for any queries or issues.

## Technologies Used
- **Frontend:** React Native for a smooth mobile experience.
- **Backend:** Node.js with Express.js to handle API requests.
- **Database:** MongoDB for flexible data storage.
- **Location Services:** Google Maps API for real-time location tracking.

## Getting Started
To get started with the FlyFood application:
1. Clone the repository:
   ```bash
   git clone https://github.com/comdputehr-ai/FlyFood.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
