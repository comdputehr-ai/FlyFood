# Tajik Eats - PRD

## Оригинальное требование
Приложение для доставки еды по всему Таджикистану. Все рестораны в одном приложении.

## Выбор пользователя
- Расширенный функционал: просмотр ресторанов, меню, корзина, оформление заказа + отслеживание заказа, история заказов, избранное
- Оплата: наличными + онлайн (Stripe)
- Расширенная админ-панель: управление меню, заказами + аналитика
- Авторизация: номер телефона или email (JWT)
- Города: Душанбе, Худжанд, Курган-Тюбе, Куляб

## Архитектура
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **Оплата**: Stripe Checkout (emergentintegrations)
- **Стилизация**: Manrope + Inter шрифты, оранжевая тема (#FF5500)

## Целевая аудитория
- Пользователи в Таджикистане, желающие заказать еду
- Владельцы ресторанов для управления бизнесом
